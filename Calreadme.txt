Step 1: open the CalculatorNode Project and install all the modules using "npm install" in the terminal.
Step 2: start the server using nodmon (if you don't nodemon just type npm app.js in the terminal)
Step 3: open the calculatorreact Project and install all the modules using "npm install" in the terminal.
step 4: start the client using npm start in the terminal
step 5: once the page opens up just type the values in the 2 textboxes and and press the button for each operation.
